# Introduction
Some basic settings to get set up with Job Set

# Variables
Set up the follwoing variables:

### Project Defaults

| Name | Description |
| --- | --- |
| PROJECTNAME | The name of the project, not sure how to set this |
| CURRENT_DATE | todays date |

### Variables I've defined
| Name | Description |
| --- | --- |
| VERSION | Issue of files being issued
| PROJECT# | i.e. PCB-001 |
| PROJECTN | Porject Name |

# Usage
${PROJECTNAME}
${CURRENT_DATE}
